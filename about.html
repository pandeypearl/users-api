Users API

This is a simple mock users API intended for front end develeopers like myself who would need mock users for your projects. You can interct with with this data using GET, POST, PUT, PATCH, and DELETE. Please note there is no database for this API so your POST, PUT, PATCH and DELETE will not be saved.

As an example you can interact with the api with all the data:

                        
                            fetch('https://users-api-mexb.onrender.com/users')
                                .then(res => {
                                    return res.json();
                                })
                                .then(data => {
                                    /* Your code here */
                                })
Or with a specific user's data: 
                        
                            fetch('https://users-api-mexb.onrender.com/users/5')
                                .then(res => {
                                    return res.json();
                                })
                                .then(data => {
                                    /* Your Code here */
                                })

I've displayed all the users indivually on the home page if you would like to interact with one or many specific users.The users Id's start from 001 as shown on the user card so if you're looking to interact with the first user with the id of 001 the user would be 0, the second user with an Id of 002 would be 1 and so on
